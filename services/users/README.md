# Nestlancer Users Service

Manages user profiles, preferences, sessions, 2FA settings, avatars, and data exports.

## Service Features
- User profile & preferences management
- Session tracking and remote termination
- 2FA Setup (TOTP configuration)
- Account activity logs
- Admin user management
- Avatar upload/management

## Commands
```bash
# Start locally
pnpm run start:dev

# Run tests
pnpm test
```
