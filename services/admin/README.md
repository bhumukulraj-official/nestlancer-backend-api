# Admin Service

The Admin Service for the Nestlancer platform, responsible for:
- System Configuration (key/value store)
- Dashboard Metrics and Aggregations
- User Impersonation Sessions
- Audit Tracking
- Database Backup Management
- Webhook Delivery Management

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
