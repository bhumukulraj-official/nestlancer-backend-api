# Nestlancer Requests Service

Manages project requests from clients, driving a state machine flow from `DRAFT` to `SUBMITTED`, `REVIEW`, `QUOTED`, and optionally creating quotes.

## Service Features
- Request creation & state management
- Attachment management via Storage
- Request timeline tracking
- Administrative oversight & quote creation

## Commands
```bash
# Start locally
pnpm run start:dev

# Run tests
pnpm test
```
